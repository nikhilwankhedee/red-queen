# Notebooks

This folder contains Jupyter notebooks for experimentation, data analysis, and model development.

## Usage

Place your `.ipynb` notebooks here for:
- Data exploration
- Model training experiments
- Visualization and analysis
- Prototyping new features

## Guidelines

- Keep notebooks well-documented with markdown cells
- Name notebooks with a number prefix for ordering (e.g., `01_data_exploration.ipynb`)
- Clean outputs before committing to keep repo size manageable
